# Panorama-Client

> **Shoot! Take a Panorama**

**`PANORAMA`** : 전경,풍경들을 연속적으로 보여주는 장치, 연속적인 광경

**`PANORAMA`** 프로젝트란? GSM(광주소프트웨어마이스터고등학교)학생들의 프로젝트를 전시하는 목적으로 개발된 웹페이지입니다.
